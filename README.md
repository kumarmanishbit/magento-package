# Project Info

## Installation

You can install the package via composer:

```bash
composer require bronto/magento-package
```

## Usage (Just Like)

```php
$firstClass = new Bronto\StarterPackage\FirstClass();
echo $firstClass->output('Hey You Done a Great Job!');
```
